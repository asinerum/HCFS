# HCFS
HCFS: Home Compact File Server
## Introduction
HCFS, standing for Home Compact File Server, is a part of the [RYOS](https://github.com/asinerum/RYOS) international toy project.  

You people are about to turn your very old Android smartphones into productive compact file servers at home, and publish them over the internet via tunnel services such as CloudFlare Tunnels, Zrok, Ngrok, Pinggy, etc.  

Your family and community members, your close friends, your colleagues and students, all could have been your HCFS users, at almost no cost.  

So don't miss the things.
